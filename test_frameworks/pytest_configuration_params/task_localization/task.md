<h2>Задание: запуск автотестов для&nbsp;разных языков&nbsp;интерфейса</h2>

<p>Мы хотим, чтобы разрабатываемый нами интернет-магазин работал одинаково хорошо для пользователей из любой страны.
Чтобы убедиться в работоспособности решения с поддержкой разных языков,
мы планируем запускать набор автотестов для каждого языка. 
Вам как разработчику автотестов нужно реализовать решение, которое позволит запускать автотесты для разных языков пользователей,
передавая нужный язык в командной строке при запуске тестов.</p>

<ol>
	<li>Добавьте в&nbsp;файл&nbsp;conftest.py обработчик, который считывает из командной строки параметр language.</li>
	<li>Реализуйте в файле conftest.py логику&nbsp;запуска браузера с указанным&nbsp;языком пользователя. Браузер должен объявляться в фикстуре browser&nbsp;и передаваться в тест как параметр.</li>
	<li>В файл test_items.py напишите тест со следующим сценарием: 
        <ul> - Открыть страницу с товаром <a href="http://selenium1py.pythonanywhere.com/catalogue/coders-at-work_207/" rel="noopener noreferrer nofollow">http://selenium1py.pythonanywhere.com/catalogue/coders-at-work_207/</a></ul>
        <ul> - Найти кнопку добавление в корзину</ul>
        <ul> - Проверить что после нажатия на кнопку появляется элемент, сигнализирующий об успешном добавлении в корзину</ul> </li>
	<li>Тест должен запускаться&nbsp;с параметром language следующей командой:
	<pre>
<code class="language-bash">pytest --language=es test_items.py</code></pre>
	и проходить успешно. Достаточно, чтобы код работал только для браузера Сhrome.</li>
