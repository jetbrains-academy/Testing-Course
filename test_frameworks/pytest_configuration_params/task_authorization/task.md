<h2>Task: authorization on a web site</h2>

<p>This task requires an account on the teaching platform <a href="https://stepik.org/" rel="noopener noreferrer nofollow"> https://stepik.org/ </a> .

Please register your personal account there with an email and a password. </p>
<p> Then you need to implement a script that will authorize you in the stepik interface on your behalf.</p>
<p>

Please be careful and do not add your login and password to public repositories on GitHub.</p>

<p>Your task is to implement an automated test with the following steps:</p>

<ul>
	<li>Open the link in Chrome: <a href="https://stepik.org/lesson/236895/step/1" rel="noopener noreferrer nofollow">https://stepik.org/lesson/236895/step/1</a>.</li>
	<li>Authorise with your own credentials.</li>
	<li>Wait until the log-in dialog disappears.</li>
</ul>

<p> Remember all the information from the second and third section. When your test is executed successfully, proceed with the next task./ </p>
