<h2>Задание: работа с выпадающим списком</h2>

<p>Для этой задачи мы придумали еще один вариант капчи для роботов. Придется немного переобучить нашего робота, чтобы он справился с новым заданием.</p>

<p>Напишите код, который реализует следующий сценарий:</p>

<ol>
	<li>Открыть страницу <a href="http://suninjuly.github.io/selects1.html" rel="nofollow noopener noreferrer" title="Link: http://suninjuly.github.io/selects1.html">https://suninjuly.github.io/selects1.html</a></li>
	<li>Посчитать сумму заданных чисел</li>
	<li>Выбрать в выпадающем списке значение равное расчитанной сумме</li>
	<li>Нажать кнопку "Submit"</li>
</ol>

<p>Если все сделано правильно и достаточно быстро (в этой задаче тоже есть ограничение по времени), вы увидите окно с числом. Отправьте полученное число в качестве ответа для этого задания.</p>

<p> </p>

<p>Когда ваш код заработает, попробуйте запустить его на странице <a href="http://suninjuly.github.io/selects2.html" rel="nofollow noopener noreferrer">https://suninjuly.github.io/selects2.html</a>. Ваш код и для нее тоже ﻿должен пройти успешно.</p>
