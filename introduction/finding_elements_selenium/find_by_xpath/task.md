<h2>Задание: поиск элемента по XPath</h2>

<p>На этот раз воспользуемся возможностью искать элементы по XPath.&nbsp;</p>

<p>На странице <a href="http://suninjuly.github.io/find_xpath_form" rel="noopener noreferrer nofollow">http://suninjuly.github.io/find_xpath_form</a>&nbsp;вы найдете такую же форму регистрации, как в шаге 3, при этом&nbsp;в нее добавилась&nbsp;куча одинаковых кнопок отправки. Но сработает только кнопка с текстом &quot;Submit&quot;, и наша задача нажать в коде именно на неё.&nbsp;</p>

<p>Ваши шаги:&nbsp;</p>

<ol>
	<li>В коде из <a href="/lesson/138920/step/4" rel="noopener noreferrer nofollow">шага 4</a>&nbsp;замените ссылку на&nbsp; <a href="http://suninjuly.github.io/find_xpath_form" rel="noopener noreferrer nofollow">http://suninjuly.github.io/find_xpath_form</a>.</li>
	<li>Подберите уникальный XPath-селектор так, чтобы он находил <em>только</em> кнопку с текстом <strong>Submit</strong>. XPath можете формулировать как угодно (по тексту, по структуре, по атрибутам) -&nbsp;главное, чтобы он работал.</li>
	<li>Модифицируйте код из шага 3 таким образом, чтобы поиск кнопки происходил с помощью XPath.</li>
	<li>Запустите ваш код.</li>
</ol>

<p>Если вы подобрали правильный селектор и все прошло хорошо, то вы получите код, который нужно отправить в качестве ответа на это задание.</p>
